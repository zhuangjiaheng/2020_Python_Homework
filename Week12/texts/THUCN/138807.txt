《盗梦空间》9月2日上映 预热手册提前热身(图)
　　警告：这不是一部轻松的电影。它可能会让你很爽，也可能会让你后悔浪费时间。《盗梦空间》里出现了大量关于“梦”的场景，但很多时候梦与现实混在一起，难于分辨。它需要思索和逻辑判断，或者这根本不适合你；或许它药性过强，而你恰恰对此过敏。为此我们准备了这个心理皮试。强烈建议认真作答后根据结果再决定是否翻开这一页。
　　1.《黑客帝国》中的锡安是：
　　A 另一个虚拟世界 B 真实存在的 C 锡安是什么？没印象
　　2.你在什么时候发现《第六感》真相的？
　　A 看到半小时左右 B 看到影片中间 C 快到结尾时
　　3.你认为梦是？
　　A 内心愿望的体现 B 日有所思，夜有所梦 C 一种预兆
　　4.《蝴蝶效应》看得你
　　A 很带劲 B 有点乱 C 很头疼
　　5.《穆赫兰道》究竟是怎么回事：
　　A 黛安杀了卡米拉，她疯了 B 黛安、丽塔、贝蒂、卡米拉是同一个人 C 贝蒂帮助丽塔逃开追杀，她们相爱了
　　6.《全面回忆》中，施瓦辛格怎样辨别出他的妻子和医生对他说谎，其实他并非在梦中的？
　　A 他残留着真实世界中的记忆 B 维持幻觉的药物失效 C 医生流汗了
　　7.《黑客帝国2》中，莫妮卡·贝鲁奇为什么非要吻基努·里维斯？
　　A 喜欢他 B 偷取the one的代码 C 让他的爱人崔妮蒂和自己的丈夫嫉妒
　　8.《死亡幻觉》中，为什么那个骑车的女孩后来不认识男主角了？
　　A 因为只在另一个时空里她才认识他 B 她装作不认识 C 她的记忆被抹去了
　　9.《记忆碎片》里主人公的妻子到底怎么了？
　　A 他没有妻子，全是幻想的 B 他的妻子在新婚之夜被歹徒杀害了 C 是他自己把妻子杀了
　　10.诺兰的《魔道争锋》里，休·杰克曼那台机器究竟是干什么用的？
　　A 穿越时空 B 复制物质 C 瞬间移动
　　计分方法：
　　1-5题选A得3分，选B得2分，选C得0分。
　　6-10题正确答案依次为C、B、A、A、B
　　结果分析：
　　24—30分：《盗梦空间》一定是你最喜欢的那种电影，不必担心，尽情期待吧！
　　15—24分(不含)：在看片的时候一定要盯紧了，或者你需要看至少两次。
　　15分以下：是否继续往下读？考虑一下吧。
